Atualmente, a aplicação está estruturada no modelo monolítico, ou seja, todos os componentes estão integrados e executando em um único ambiente.

🎯 Objetivo

O objetivo é realizar a containerização da aplicação e reestruturá-la em uma arquitetura baseada em microserviços.



⚙️ Requisitos da atividade
Você deverá organizar a aplicação da seguinte forma:

1 container para o Front-end + Back-end
1 container para o SGBD (phpMyAdmin)
1 container para o Banco de Dados (MySQL)
📌 Tecnologias envolvidas

phpMyAdmin como SGBD
MySQL como banco de dados
